State definition for nodes.

Node is starting registration by ClientHello message

WAIT_FOR_SERVER_HELLO 

Node sent ClientHello message, now it is waiting for ServerHello

WAIT_FOR_DOWNLINK 

Node sent a data message, now it is waiting for downlink data

REGISTERED 

Node is registered and its key is valid

SLEEP 

Node is in sleep mode
